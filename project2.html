E-Commerce Platform

Project Overview

A full-stack e-commerce platform built using .NET for the backend and modern frontend technologies. The platform supports user registration, product browsing, shopping cart functionality, and secure payment processing.

Technologies Used

  • .NET Core (C#)
  • ASP.NET MVC
  • Entity Framework
  • SQL Server
  • HTML, CSS, JavaScript

Features

  • User authentication and roles
  • Product catalog with categories
  • Shopping cart and checkout
  • Order history and tracking
